1. What is the difference between local storage and session storage?

Ans: The difference between sessionStorage and localStorage is that localStorage data does not expire, whereas sessionStorage data is cleared when the page session ends

2. What is the difference between global and block scope?

Ans: Variables declared with var, let and const are quite similar when declared outside a block.They all have Global Scope .Block scope is variables declared inside a { } block cannot be accessed from outside the block.

3. How does js event loop work?

Ans: Event loop is an endless loop, which waits for tasks, executes them and then sleeps until it receives more tasks. The event loop executes tasks from the event queue only when the call stack is empty i.e. there is no ongoing task. The event loop allows us to use callbacks and promises.
